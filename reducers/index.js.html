¶ Imports

our 4 reducers

 
import auth from './auth/authReducer';

¶ CombineReducers

the rootReducer will call each and every reducer with the state and action EVERY TIME there is a basic action

 
const rootReducer = combineReducers({
  auth,
  device,
  global,
  profile
});

export default rootReducer;
